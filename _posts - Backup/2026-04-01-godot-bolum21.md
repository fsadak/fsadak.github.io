# Squash the Creeps! — Bölüm 5: Düşmanları Oluşturmak (Spawn)

Mob sahnesi hazır. Şimdi düşmanları oyun alanına rastgele spawn etme zamanı! Bu bölümde kamera görüş alanının etrafında bir spawn yolu çizecek ve Timer ile her 0.5 saniyede bir düşman oluşturacağız.

---

## Pencere Çözünürlüğünü Ayarla

Önce oyun penceresini küçültelim. `Project > Project Settings` menüsünü aç:

![Proje Ayarları](/assets/images/02.project_settings.webp)
*Project Settings'i açıyoruz*

**Display > Window** bölümüne git ve şu değerleri ayarla:

![Pencere Ayarları](/assets/images/03.window_settings.webp)
*Viewport Width: 720, Viewport Height: 540*

- **Viewport Width:** `720`
- **Viewport Height:** `540`

2D oyunda `Path2D` kullandık. 3D'de de benzer şekilde `Path3D` ve `PathFollow3D` kullanacağız. Ancak 3D'de yolu çizerken kamera önizlemesinde göremeyiz. Yolun nereye gideceğini anlamak için **yer tutucu silindirler** kullanacağız.

Viewport hâlâ ikiye bölünmüş olmalı (Ctrl+2 ile açabilirsin). Alt viewport'ta kamera önizlemesini aç:

![Kamera Önizlemesi](/assets/images/04.camera_preview.png)
*Alt viewport'ta kamera önizlemesi — spawn yolunun sınırlarını buradan takip edeceğiz*

## Main Scriptini Yaz

**Main** node'una sağ tıkla ve bir script ekle:

```gdscript
extends Node

@export var mob_scene: PackedScene
```

*timeout() sinyali Main node'una bağlandı*

### `_on_mob_timer_timeout()` Fonksiyonu

Oluşan boş fonksiyonu şu kodla doldur:

```gdscript
func _on_mob_timer_timeout():
    # Mob sahnesinden yeni bir instance oluştur
    var mob = mob_scene.instantiate()

    # SpawnPath üzerinde rastgele bir konum seç
    var mob_spawn_location = get_node("SpawnPath/SpawnLocation")
    mob_spawn_location.progress_ratio = randf()

    # Oyuncunun konumunu al
    var player_position = $Player.position

    # Mob'u başlat (konum ve oyuncu konumu ile)
    mob.initialize(mob_spawn_location.position, player_position)

    # Mob'u Main sahnesine ekle
    add_child(mob)
```

**Kodun mantığı:**

- `randf()` — 0 ile 1 arasında rastgele bir değer üretir
- `progress_ratio` — `PathFollow3D`'nin yol üzerindeki konumunu 0 (başlangıç) ile 1 (bitiş) arasında belirler
- Yolumuz kamera görüş alanının etrafını döndüğünden, 0-1 arasında herhangi bir değer ekranın kenarlarında rastgele bir konuma karşılık gelir
- `mob.initialize()` — bir önceki bölümde yazdığımız fonksiyon; mob'u konumlandırır, oyuncuya döndürür ve rastgele hız atar

## Tam Script

```gdscript
extends Node

@export var mob_scene: PackedScene

func _on_mob_timer_timeout():
    var mob = mob_scene.instantiate()
    var mob_spawn_location = get_node("SpawnPath/SpawnLocation")
    mob_spawn_location.progress_ratio = randf()
    var player_position = $Player.position
    mob.initialize(mob_spawn_location.position, player_position)
    add_child(mob)
```

## Özet

| Öğe | Açıklama |
|---|---|
| Yer tutucu silindirler | Spawn yolunun köşelerini belirlemek için görsel referans |
| `Path3D` (SpawnPath) | Spawn yolu — kamera görünümünü çerçeveler |
| `PathFollow3D` (SpawnLocation) | Yol üzerinde `progress_ratio`'ya göre rastgele konum |
| `Timer` (MobTimer) | Her 0.5 saniyede `timeout()` tetikler |
| `randf()` | 0-1 arası rastgele float üretir |
| `mob_scene.instantiate()` | Mob şablonundan yeni instance oluşturur |
