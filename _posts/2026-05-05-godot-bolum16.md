## Üye Değişkenleri Tanımla

Scriptin başına şu değişkenleri ekle:

```gdscript
extends Area2D

@export var speed = 400  # Oyuncunun hareket hızı (piksel/saniye)
var screen_size           # Oyun penceresinin boyutu
```

`speed` değişkeninin önündeki **`@export`** anahtar kelimesi, bu değeri Inspector'dan ayarlayabilmeni sağlar — yerleşik node özelliklerini ayarlar gibi.

![Export Değişken](/assets/images/export_variable.webp)
*@export sayesinde speed değişkeni Inspector'da görünür*

> ⚠️ **Dikkat:** Inspector'da değeri değiştirirsen, scriptteki değerin üzerine yazılır.

## `_ready()` ile Ekran Boyutunu Al

`_ready()` fonksiyonu, node sahne ağacına girdiğinde çağrılır. Oyun penceresinin boyutunu burada alacağız:

```gdscript
func _ready():
    screen_size = get_viewport_rect().size
```

## `_process()` ile Hareketi Kodla

`_process()` fonksiyonunu şu şekilde yaz:

```gdscript
func _process(delta):
    var velocity = Vector2.ZERO  # Oyuncunun hareket vektörü

    if Input.is_action_pressed("move_right"):
        velocity.x += 1
    if Input.is_action_pressed("move_left"):
        velocity.x -= 1
    if Input.is_action_pressed("move_down"):
        velocity.y += 1
    if Input.is_action_pressed("move_up"):
        velocity.y -= 1

    if velocity.length() > 0:
        velocity = velocity.normalized() * speed
        $AnimatedSprite2D.play()
    else:
        $AnimatedSprite2D.stop()

    position += velocity * delta
    position = position.clamp(Vector2.ZERO, screen_size)
```

### Kodun Mantığı

**1. Hız vektörü başlatma:**
`velocity = Vector2.ZERO` ile başlıyoruz — varsayılan olarak oyuncu hareket etmiyor.

**2. Tuş kontrolleri:**
Basılan her tuşa göre vektöre ekleme veya çıkarma yapıyoruz. Sağ + Aşağı aynı anda basılırsa vektör `(1, 1)` olur.

**3. Diagonal hareketi düzeltme:**
`(1, 1)` vektörünün uzunluğu `√2 ≈ 1.41`'dir — bu, yatay/dikey hareketten %41 daha hızlı demek! Bunu önlemek için `velocity.normalized()` ile vektörün uzunluğunu 1'e indirip `speed` ile çarpıyoruz.

**4. Animasyon:**
Hareket varsa `play()`, yoksa `stop()` çağırıyoruz.

**5. Konum güncelleme:**
`position += velocity * delta` ile oyuncuyu hareket ettiriyoruz. `delta` kullanmak, hareketi kare hızından bağımsız kılıyor.

**6. Ekran sınırlarına kilitleme:**
`position.clamp(Vector2.ZERO, screen_size)` oyuncunun ekran dışına çıkmasını engelliyor.

> 💡 **İpucu:** `$AnimatedSprite2D`, `get_node("AnimatedSprite2D")` için GDScript kısayoludur.

**F6** ile sahneyi çalıştır ve ok tuşlarıyla oyuncunun tüm yönlerde hareket ettiğini doğrula.

## Animasyonları Seç

Oyuncu hareket edebiliyor ama animasyon henüz yöne göre değişmiyor. `_process()` fonksiyonunun sonuna şu kodu ekle:

```gdscript
    if velocity.x != 0:
        $AnimatedSprite2D.animation = "walk"
        $AnimatedSprite2D.flip_v = false
        $AnimatedSprite2D.flip_h = velocity.x < 0
    elif velocity.y != 0:
        $AnimatedSprite2D.animation = "up"
        $AnimatedSprite2D.flip_v = velocity.y > 0
```

### Animasyon Mantığı

- **Sağa giderken:** `walk` animasyonu, yatay çevirme yok
- **Sola giderken:** `walk` animasyonu, `flip_h = true` ile yatay çevrilmiş
- **Yukarı giderken:** `up` animasyonu, dikey çevirme yok
- **Aşağı giderken:** `up` animasyonu, `flip_v = true` ile dikey çevrilmiş

> 📝 **Boolean atama notu:** `$AnimatedSprite2D.flip_h = velocity.x < 0` ifadesi hem karşılaştırma hem de atama yapıyor. `velocity.x < 0` ifadesi `true` veya `false` üretir ve doğrudan `flip_h`'ye atanır. Uzun yazımı şöyle olurdu:
> ```gdscript
> if velocity.x < 0:
>     $AnimatedSprite2D.flip_h = true
> else:
>     $AnimatedSprite2D.flip_h = false
> ```

Sahneyi tekrar çalıştır ve her yönde animasyonların doğru çalıştığını kontrol et.

> ⚠️ **Dikkat:** Animasyon isimlerini doğru yazmaya özen göster. `SpriteFrames` panelinde `"walk"` olarak adlandırdıysan kodda da `"walk"` yazmalısın — büyük-küçük harf duyarlıdır!

## Oyun Başladığında Oyuncuyu Gizle

Oyun başladığında oyuncunun gizli olmasını istiyoruz. `_ready()` fonksiyonuna şu satırı ekle:

```gdscript
func _ready():
    screen_size = get_viewport_rect().size
    hide()
```

## Çarpışmaya Hazırlık: Özel Sinyal Tanımla

Düşmanlarla çarpışmayı tespit etmek için bir **özel sinyal** tanımlayacağız. Scriptin en üstüne, `extends Area2D` satırından hemen sonra ekle:

```gdscript
signal hit
```

Bu, oyuncunun düşmanla çarpıştığında yayacağı `hit` sinyalini tanımlar.

Şimdi **Signals** sekmesinde (Inspector'ın yanında) oyuncunun sinyallerini kontrol et:

![Oyuncu Sinyalleri](/assets/images/player_signals.webp)
*Signals sekmesinde özel "hit" sinyalimiz de görünüyor*

Düşmanlarımız `RigidBody2D` node'u olacağından `body_entered(body: Node2D)` sinyalini kullanacağız. Bu sinyal, bir gövde oyuncuya temas ettiğinde tetiklenir. **"Connect..."** butonuna tıkla:

![Sinyal Bağlantısı](/assets/images/player_signal_connection.webp)
*body_entered sinyalini Player script'ine bağlıyoruz*

Godot, scriptte otomatik olarak `_on_body_entered` fonksiyonunu oluşturur. Bu fonksiyona şu kodu ekle:

```gdscript
func _on_body_entered(_body):
    hide()                                          # Oyuncu vurulunca kaybolur
    hit.emit()                                      # hit sinyalini yayıyoruz
    # Fizik callback sırasında özelliği değiştirmek hata verebilir,
    # bu yüzden set_deferred kullanıyoruz
    $CollisionShape2D.set_deferred("disabled", true)
```

Bir düşman oyuncuya her çarptığında:
1. Oyuncu gizlenir
2. `hit` sinyali yayılır
3. Çarpışma şekli devre dışı bırakılır — böylece sinyal birden fazla kez tetiklenmez

> 💡 **Neden `set_deferred()`?** Çarpışma şeklini fizik işleme sırasında doğrudan devre dışı bırakmak hata verebilir. `set_deferred()`, Godot'ya işlemin güvenli olduğu zamana kadar beklemesini söyler.

## Yeni Oyun Başlatmak İçin `start()` Fonksiyonu

Son olarak, yeni bir oyun başlatıldığında oyuncuyu sıfırlamak için bir fonksiyon ekle:

```gdscript
func start(pos):
    position = pos
    show()
    $CollisionShape2D.disabled = false
```

Bu fonksiyon oyuncuyu verilen konuma yerleştirir, görünür yapar ve çarpışma şeklini yeniden etkinleştirir.

## Tam Script

```gdscript
extends Area2D

signal hit

@export var speed = 400
var screen_size

func _ready():
    screen_size = get_viewport_rect().size
    hide()

func _process(delta):
    var velocity = Vector2.ZERO

    if Input.is_action_pressed("move_right"):
        velocity.x += 1
    if Input.is_action_pressed("move_left"):
        velocity.x -= 1
    if Input.is_action_pressed("move_down"):
        velocity.y += 1
    if Input.is_action_pressed("move_up"):
        velocity.y -= 1

    if velocity.length() > 0:
        velocity = velocity.normalized() * speed
        $AnimatedSprite2D.play()
    else:
        $AnimatedSprite2D.stop()

    position += velocity * delta
    position = position.clamp(Vector2.ZERO, screen_size)

    if velocity.x != 0:
        $AnimatedSprite2D.animation = "walk"
        $AnimatedSprite2D.flip_v = false
        $AnimatedSprite2D.flip_h = velocity.x < 0
    elif velocity.y != 0:
        $AnimatedSprite2D.animation = "up"
        $AnimatedSprite2D.flip_v = velocity.y > 0

func _on_body_entered(_body):
    hide()
    hit.emit()
    $CollisionShape2D.set_deferred("disabled", true)

func start(pos):
    position = pos
    show()
    $CollisionShape2D.disabled = false
```

## Özet

| Özellik | Nasıl Yapıldı? |
|---|---|
| Hareket | `Input.is_action_pressed()` + `velocity` vektörü |
| Diagonal düzeltme | `velocity.normalized() * speed` |
| Ekran sınırı | `position.clamp()` |
| Animasyon seçimi | `velocity.x` ve `velocity.y`'ye göre `animation` ve `flip` |
| Çarpışma tespiti | `body_entered` sinyali |
| Çarpışma devre dışı | `set_deferred("disabled", true)` |
