# Squash the Creeps! — Bölüm 6: Zıplama ve Düşmanları Ezme

Düşmanlar spawn oluyor ama oyuncu henüz zıplayamıyor ve düşmanları ezemıyor. Bu bölümde önce fizik katmanlarını ayarlayacak, sonra zıplama ve ezme mekaniğini kodlayacağız.

---

## Fizik Katmanlarını Anlama

Godot'daki fizik gövdeleri iki tamamlayıcı özelliğe sahiptir: **layer** ve **mask**.

- **Layer:** Nesnenin hangi fizik katmanında olduğunu tanımlar
- **Mask:** Nesnenin hangi katmanları "dinleyeceğini" — yani hangileriyle çarpışma algılayacağını — belirler

İki nesnenin etkileşebilmesi için en az birinin, diğerinin katmanına karşılık gelen bir maskesi olması gerekir. Bu sistem sayesinde hangi nesnelerin birbirleriyle çarpışacağını, hangilerinin çarpışmayacağını kolayca kontrol edebilirsin.

Varsayılan olarak tüm fizik gövdeleri hem layer hem mask olarak **1** atanmıştır — hepsi birbirleriyle çarpışır.

### Player

`player.tscn`'yi aç. **Player** node'unu seç:
- **Layer:** olduğu gibi bırak (1 = `player` katmanı)
- **Mask:** `enemies` ve `world`'ü aç

![Oyuncu Fizik Maskesi](/assets/images/07.player_physics_mask.webp)
*Player: Layer = player, Mask = enemies + world*

### Mob

`mob.tscn`'yi aç. **Mob** node'unu seç:
- **Layer:** `enemies`'i aç, 1'i kapat
- **Mask:** tamamen kapat (mob'lar birbirlerine ve zemine çarpmıyor)

![Mob Fizik Maskesi](/assets/images/08.mob_physics_mask.webp)
*Mob: Layer = enemies, Mask = boş*

> 💡 Mob'ların birbirleriyle çarpışmasını istersen `enemies` maskesini aç. Şu an birbirlerinden geçiyorlar — bu tasarım gereği.

## Zıplama Mekaniği

Zıplama için sadece iki satır kod yeterli! Player scriptini aç.

Değişkenler bölümüne `jump_impulse` ekle:

```gdscript
# Zıplarken karaktere uygulanacak dikey itme kuvveti (m/s)
@export var jump_impulse = 20
```

`_physics_process()` içinde `move_and_slide()` çağrısından **önce** şu kodu ekle:

```gdscript
    # Zıplama
    if is_on_floor() and Input.is_action_just_pressed("jump"):
        target_velocity.y = jump_impulse
```

Bu kadar! `is_on_floor()` zemine temas olup olmadığını kontrol eder. Eğer yerdeyken "jump" tuşuna basılırsa, anında büyük bir dikey hız verilir.

> 💡 **3D'de Y ekseni yukarıya pozitif!** 2D'den farklıdır — 2D'de Y aşağıya pozitiftir.

## Ezme Mekaniğini Kodla

### Player Scriptine `bounce_impulse` Ekle

```gdscript
# Bir mob'un üstüne basıldığında uygulanacak dikey itme (m/s)
@export var bounce_impulse = 16
```

### Ezme Döngüsü

`_physics_process()` içinde zıplama kodunun **ardından** şu döngüyü ekle:

```gdscript
    # Bu karede gerçekleşen tüm çarpışmaları tara
    for index in range(get_slide_collision_count()):
        var collision = get_slide_collision(index)

        # Null çarpışma kontrolü (aynı karede iki kez çarparsa)
        if collision.get_collider() == null:
            continue

        # Eğer bir mob'a çarptıysak
        if collision.get_collider().is_in_group("mob"):
            var mob = collision.get_collider()
            # Yukarıdan mı çarpıyoruz kontrol et
            if Vector3.UP.dot(collision.get_normal()) > 0.1:
                # Mob'u ez ve zıpla
                mob.squash()
                target_velocity.y = bounce_impulse
                # Aynı mob'u iki kez işlemeyi önle
                break
```

### Yeni Fonksiyonları Anlamak

**`get_slide_collision_count()` ve `get_slide_collision()`:**
`move_and_slide()`, hareketi bazen birden fazla adımda gerçekleştirir. Bu nedenle tüm çarpışmaları döngüyle kontrol etmemiz gerekiyor.

**`is_in_group("mob")`:**
Her Node'da kullanılabilir; node'un belirtilen grupta olup olmadığını döner.

**`Vector3.UP.dot(collision.get_normal()) > 0.1`:**
Dot (nokta) çarpımı iki vektör arasındaki açıyı ölçer. `collision.get_normal()` çarpışma düzlemine dik vektördür. Sonuç 0.1'den büyükse yukarıdan — yani mob'un **üstüne** — çarpıyoruz demektir.

**`break`:**
Aynı mob'u birden fazla kez işlemeyi önler — skor birden fazla sayılmaz.

## Mob Scriptine `squash()` Fonksiyonu Ekle

Player, `mob.squash()` çağırıyor ama bu fonksiyon henüz yok. `mob.gd`'yi aç.

Scriptin başına yeni bir sinyal ekle:

```gdscript
# Oyuncu mob'un üstüne atladığında tetiklenir
signal squashed
```

Scriptin sonuna `squash()` fonksiyonunu ekle:

```gdscript
func squash():
    squashed.emit()
    queue_free()
```

`squashed` sinyalini bir sonraki bölümde skoru artırmak için kullanacağız.

## Güncel Tam Scriptler

### player.gd

```gdscript
extends CharacterBody3D

signal squashed

@export var speed = 14
@export var fall_acceleration = 75
@export var jump_impulse = 20
@export var bounce_impulse = 16

var target_velocity = Vector3.ZERO

func _physics_process(delta):
    var direction = Vector3.ZERO

    if Input.is_action_pressed("move_right"):
        direction.x += 1
    if Input.is_action_pressed("move_left"):
        direction.x -= 1
    if Input.is_action_pressed("move_back"):
        direction.z += 1
    if Input.is_action_pressed("move_forward"):
        direction.z -= 1

    if direction != Vector3.ZERO:
        direction = direction.normalized()
        $Pivot.basis = Basis.looking_at(direction)

    # Zıplama
    if is_on_floor() and Input.is_action_just_pressed("jump"):
        target_velocity.y = jump_impulse

    # Ezme
    for index in range(get_slide_collision_count()):
        var collision = get_slide_collision(index)
        if collision.get_collider() == null:
            continue
        if collision.get_collider().is_in_group("mob"):
            var mob = collision.get_collider()
            if Vector3.UP.dot(collision.get_normal()) > 0.1:
                mob.squash()
                target_velocity.y = bounce_impulse
                break

    target_velocity.x = direction.x * speed
    target_velocity.z = direction.z * speed

    if not is_on_floor():
        target_velocity.y = target_velocity.y - (fall_acceleration * delta)

    velocity = target_velocity
    move_and_slide()
```

### mob.gd (güncellendi)

```gdscript
extends CharacterBody3D

signal squashed

@export var min_speed = 10
@export var max_speed = 18

func _physics_process(_delta):
    move_and_slide()

func initialize(start_position, player_position):
    look_at_from_position(start_position, player_position, Vector3.UP)
    rotate_y(randf_range(-PI / 4, PI / 4))
    var random_speed = randi_range(min_speed, max_speed)
    velocity = Vector3.FORWARD * random_speed
    velocity = velocity.rotated(Vector3.UP, rotation.y)

func squash():
    squashed.emit()
    queue_free()

func _on_visible_on_screen_notifier_3d_screen_exited():
    queue_free()
```

## Özet

| Kavram | Açıklama |
|---|---|
| **Fizik Katmanları** | Nesnelerin hangi fizik katmanında olduğunu belirler |
| **Fizik Maskeleri** | Hangi katmanlarla çarpışma algılanacağını belirler |
| `is_action_just_pressed()` | Tuşa **bu karede** basıldıysa true (sürekli basılı değil) |
| `get_slide_collision()` | `move_and_slide()` sırasındaki çarpışma bilgisi |
| `is_in_group()` | Node'un belirtilen grupta olup olmadığını kontrol eder |
| Dot Product | İki vektör arasındaki açıyı ölçer — yukarıdan çarpma tespiti |
